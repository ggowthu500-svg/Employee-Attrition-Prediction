# Employee Attrition Prediction

## Overview
Employee Attrition Prediction is a Machine Learning project that predicts whether an employee is likely to leave the company or stay. The application uses the IBM HR Analytics Employee Attrition dataset and a Random Forest Classifier to provide accurate predictions. A simple Gradio web interface allows users to enter employee details and instantly view the prediction result.

---

## Features
- Predict employee attrition (Leave/Stay)
- Interactive Gradio web interface
- Data preprocessing using Label Encoding
- Random Forest Machine Learning model
- Easy-to-use prediction system
- Fast and accurate classification

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Employee-Attrition-Prediction.git
```

2. Navigate to the project folder

```bash
cd Employee-Attrition-Prediction
```

3. Install the required libraries

```bash
pip install pandas numpy scikit-learn gradio
```
   ## Run the Project

1. Open `Employee_Attrition_Prediction.ipynb` in Google Colab.
2. Upload the dataset `WA_Fn-UseC_-HR-Employee-Attrition.csv`.
3. Run all cells in the notebook.
4. Enter employee details in the Gradio interface.
5. View the prediction result.
